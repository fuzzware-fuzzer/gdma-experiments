AT Command Example
========
This example application is a simple command executor.
The source is base on cAT [1] and is build with the mcux-sdk [2].
It uses the FRDM-K64F as target borad.
Communication is done via uart with DMA.

[1] https://github.com/marcinbor85/cAT
[2] https://github.com/nxp-mcuxpresso/mcux-sdk
